# 자기소개서 작성 프로젝트

## 개요

자기소개서를 텍스트로 저장하니 외부에서 작성 및 참고가 힘들고, 여러 사이트를 이용하니 내가 원하는 회사는 안올라와 있는 경우도 많았다.

그래서 이를 해결하고자 만든 자기소개서 작성 및 저장 웹 서비스.

## 기능
* 회원가입 & 로그인
* 원하는 채용 공고 생성, 삭제 및 수정
* 채용 공고안에 원하는 문항 생성, 삭제 및 수정
* 작성 중인 내용에 대한 글자 수 확인 및 맞춤법 검사

## 기술 스택

#### Back-End
* Spring Boot
* Fast API
#### Front-End
* React
